e-Granthalaya

A responsive web-based digital library platform designed to provide students with easy access to Maharashtra SSC textbooks and entrance exam preparation resources through a clean and user-friendly interface.

📚 Project Overview

e-Granthalaya is a mini web development project that acts as an online educational resource portal. The platform allows users to browse, search, and view academic books and entrance exam preparation materials directly within the website using an embedded document viewer.

The project focuses on simplicity, accessibility, and responsive user experience.

✨ Features
Responsive and modern user interface
Dynamic book catalog rendering using JavaScript
Search functionality for quick resource access
Embedded PDF/document viewer using modal popup
Categorized educational resources
Interactive card-based layout
Mobile-friendly responsive design
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Google Drive Embedded PDF Viewer
📖 Resources Included

The platform contains educational materials such as:

Maharashtra SSC Textbooks
JEE Preparation Books
NEET Preparation Materials
MHT-CET Study Resources
Solved Papers and Crash Courses
⚙️ How It Works
The application stores book data in a JavaScript array.
Books are dynamically displayed as cards on the webpage.
Users can search for books using the search bar.
Clicking on View Document opens the selected resource inside a modal document viewer.
PDFs are displayed using embedded Google Drive preview links.
